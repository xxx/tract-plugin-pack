# Port GUI from Vizia to egui Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Replace the nih_plug_vizia GUI with nih_plug_egui to reduce idle CPU usage from femtovg path tessellation (~13% → target <5%).

**Architecture:** egui's immediate-mode rendering is lighter than femtovg's retained-mode path tessellation. All custom views (WavetableView, FilterResponseView, ParamDial) are rewritten using egui's `Painter` API. The layout uses egui's built-in `Ui` composition instead of Vizia's CSS-styled VStack/HStack. Parameter binding uses `ParamSetter` directly instead of Vizia's Lens/Model/Event system.

**Tech Stack:** Rust, nih-plug, nih_plug_egui (egui + baseview + OpenGL)

### Task 3: Port the wavetable view

**Files:**
- Modify: `src/editor.rs`

Port the 3D/2D wavetable visualization using egui's `Painter` API. egui uses `Shape` objects (lines, paths, rectangles) instead of NanoVG's `vg::Path`.

Key API differences:
- `vg::Path::new()` + `line_to()` → `egui::Shape::line(points, stroke)`
- `canvas.stroke_path()` → `painter.add(shape)`
- `canvas.fill_path()` → `painter.add(Shape::convex_polygon(points, fill, stroke))`

- [ ] **Step 1: Add wavetable drawing function**

Add a function to `src/editor.rs`:

```rust
fn draw_wavetable(
    ui: &mut egui::Ui,
    params: &WavetableFilterParams,
    shared_wavetable: &Arc<std::sync::Mutex<crate::wavetable::Wavetable>>,
    show_2d: &mut bool,
) {
    let (response, painter) = ui.allocate_painter(
        egui::vec2(ui.available_width(), 220.0),
        egui::Sense::click(),
    );

    if response.clicked() {
        *show_2d = !*show_2d;
    }

    let rect = response.rect;
    // Background
    painter.rect_filled(rect, 0.0, egui::Color32::from_rgb(20, 22, 28));

    let Ok(wt) = shared_wavetable.try_lock() else { return };
    let frame_count = wt.frame_count;
    let frame_size = wt.frame_size;
    if frame_count == 0 || frame_size == 0 { return; }

    let frame_pos = params.frame_position.modulated_normalized_value();
    let current_idx = (frame_pos * (frame_count - 1) as f32).round() as usize;

    let padding = 10.0;
    let w = rect.width() - padding * 2.0;
    let h = rect.height() - padding * 2.0;
    let num_points = (w as usize).min(frame_size).max(1);

    if *show_2d {
        // 2D: interpolated frame
        let exact = frame_pos * (frame_count - 1) as f32;
        let lo = (exact.floor() as usize).min(frame_count - 1);
        let hi = (lo + 1).min(frame_count - 1);
        let frac = exact - lo as f32;

        let mut fmin = f32::INFINITY;
        let mut fmax = f32::NEG_INFINITY;
        for pi in 0..num_points {
            let si = ((pi as f32 / num_points as f32) * frame_size as f32) as usize;
            let si = si.min(frame_size - 1);
            let s = wt.frames[lo][si] * (1.0 - frac) + wt.frames[hi][si] * frac;
            fmin = fmin.min(s);
            fmax = fmax.max(s);
        }
        let range = (fmax - fmin).max(0.001);

        let points: Vec<egui::Pos2> = (0..num_points).map(|pi| {
            let t = pi as f32 / num_points as f32;
            let si = ((t * frame_size as f32) as usize).min(frame_size - 1);
            let s = wt.frames[lo][si] * (1.0 - frac) + wt.frames[hi][si] * frac;
            let norm = (s - fmin) / range;
            egui::pos2(
                rect.left() + padding + t * w,
                rect.top() + padding + h - norm * h,
            )
        }).collect();

        painter.add(egui::Shape::line(
            points,
            egui::Stroke::new(1.5, egui::Color32::from_rgba_premultiplied(79, 195, 247, 220)),
        ));
    } else {
        // 3D: all frames
        let mut gmin = f32::INFINITY;
        let mut gmax = f32::NEG_INFINITY;
        for frame in &wt.frames {
            for &s in frame { gmin = gmin.min(s); gmax = gmax.max(s); }
        }
        let range = (gmax - gmin).max(0.001);

        for frame_idx in (0..frame_count).rev() {
            let frame = &wt.frames[frame_idx];
            let depth = frame_idx as f32 / frame_count.max(1) as f32;
            let px = depth * 60.0;
            let py = -depth * 60.0;
            let alpha = if frame_idx == current_idx { 255 } else { (0.3 + (1.0 - depth) * 0.4) as u8 * 255 / 100 + 80 };

            let points: Vec<egui::Pos2> = (0..num_points).map(|pi| {
                let t = pi as f32 / num_points as f32;
                let si = ((t * frame_size as f32) as usize).min(frame_size - 1);
                let norm = (frame[si] - gmin) / range;
                egui::pos2(
                    rect.left() + padding + t * (w * 0.7) + px,
                    rect.bottom() - padding * 2.0 - norm * h * 0.4 + py,
                )
            }).collect();

            let color = if frame_idx == current_idx {
                egui::Color32::from_rgba_premultiplied(255, 200, 100, 255)
            } else {
                egui::Color32::from_rgba_premultiplied(
                    (50.0 + (1.0 - depth) * 100.0) as u8,
                    (100.0 + (1.0 - depth) * 100.0) as u8,
                    255,
                    alpha,
                )
            };

            let width = if frame_idx == current_idx { 2.5 } else { 1.2 };
            painter.add(egui::Shape::line(points, egui::Stroke::new(width, color)));
        }
    }
}
```
