# Warp Zone Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a psychedelic spectral shifting/stretching effect plugin using a phase vocoder, with LFO and envelope follower modulation, and a waveform ribbon visualization GUI.

**Architecture:** New workspace crate `warp-zone/`. Core DSP (`spectral.rs`) implements STFT analysis with phase-vocoder bin remapping for frequency shifting and harmonic stretching. Modulation system (`modulation.rs`) provides LFO (sine/tri/S&H) and RMS envelope follower. Plugin (`lib.rs`) runs stereo phase vocoder with modulation applied per-hop. GUI (`editor.rs`) uses softbuffer + tiny-skia with flowing ribbon visualization driven by per-band spectral magnitudes shared via `AtomicI32` on the params struct.

**Tech Stack:** Rust nightly, nih-plug, rustfft 6, tiny-skia-widgets, softbuffer, crossbeam (atomics only)

- [ ] **Step 3: Implement SpectralShifter STFT pipeline**

Replace the stub with the full STFT implementation. Key design points:
- Identity mode (shift=0, stretch=1.0) short-circuits `remap_bins` and copies `fft_buf` directly to `out_buf`, avoiding phase accumulation drift.
- `wrap_phase` uses modular arithmetic (not while-loop) to handle arbitrarily large phase values safely.
- `remap_bins` is a stub in this task (returns identity) — actual remapping in Task 3.

```rust
//! Phase vocoder with spectral shifting and stretching.
//!
//! STFT analysis → bin remapping (shift + stretch) → phase accumulation →
//! STFT synthesis → overlap-add.
//!
//! FFT size 4096, hop size 1024 (75% overlap, 4x redundancy), Hann window.

use rustfft::num_complex::Complex;
use rustfft::{Fft, FftPlanner};
use std::f32::consts::PI;
use std::sync::Arc;

pub struct SpectralShifter {
    fft_size: usize,
    hop_size: usize,

    fft_forward: Arc<dyn Fft<f32>>,
    fft_inverse: Arc<dyn Fft<f32>>,
    scratch: Vec<Complex<f32>>,

    analysis_window: Vec<f32>,
    synthesis_window: Vec<f32>,

    input_ring: Vec<f32>,
    output_ring: Vec<f64>,
    input_pos: usize,
    read_pos: usize,
    hop_counter: usize,

    fft_buf: Vec<Complex<f32>>,
    out_buf: Vec<Complex<f32>>,

    last_input_phase: Vec<f32>,
    accumulated_output_phase: Vec<f32>,
}

impl SpectralShifter {
    pub fn new(fft_size: usize, hop_size: usize) -> Self {
        assert!(fft_size > 0 && hop_size > 0 && fft_size >= hop_size);

        let mut planner = FftPlanner::new();
        let fft_forward = planner.plan_fft_forward(fft_size);
        let fft_inverse = planner.plan_fft_inverse(fft_size);
        let scratch_len = fft_forward
            .get_inplace_scratch_len()
            .max(fft_inverse.get_inplace_scratch_len());

        let analysis_window: Vec<f32> = (0..fft_size)
            .map(|i| 0.5 * (1.0 - (2.0 * PI * i as f32 / fft_size as f32).cos()))
            .collect();

        // COLA normalization (same approach as satch)
        let num_frames = fft_size / hop_size;
        let mut cola_check = vec![0.0_f64; hop_size];
        for frame in 0..num_frames {
            let offset = frame * hop_size;
            for p in 0..hop_size {
                let w = analysis_window[p + offset] as f64;
                cola_check[p] += w * w;
            }
        }
        let cola_factor = cola_check[0] as f32;
        let inv_cola = 1.0 / cola_factor;
        let synthesis_window: Vec<f32> = analysis_window.iter().map(|&w| w * inv_cola).collect();

        let out_ring_size = 2 * fft_size;
        let half_plus_one = fft_size / 2 + 1;

        Self {
            fft_size,
            hop_size,
            fft_forward,
            fft_inverse,
            scratch: vec![Complex::new(0.0, 0.0); scratch_len],
            analysis_window,
            synthesis_window,
            input_ring: vec![0.0; fft_size],
            output_ring: vec![0.0; out_ring_size],
            input_pos: 0,
            read_pos: 0,
            hop_counter: 0,
            fft_buf: vec![Complex::new(0.0, 0.0); fft_size],
            out_buf: vec![Complex::new(0.0, 0.0); fft_size],
            last_input_phase: vec![0.0; half_plus_one],
            accumulated_output_phase: vec![0.0; half_plus_one],
        }
    }

    pub fn latency_samples(&self) -> usize {
        self.fft_size
    }

    pub fn reset(&mut self) {
        self.input_ring.fill(0.0);
        self.output_ring.fill(0.0);
        self.input_pos = 0;
        self.read_pos = 0;
        self.hop_counter = 0;
        self.last_input_phase.fill(0.0);
        self.accumulated_output_phase.fill(0.0);
        for bin in self.fft_buf.iter_mut() {
            *bin = Complex::new(0.0, 0.0);
        }
        for bin in self.out_buf.iter_mut() {
            *bin = Complex::new(0.0, 0.0);
        }
    }

    pub fn process_sample(&mut self, input: f32, shift: f32, stretch: f32) -> f32 {
        let out_len = self.output_ring.len();

        self.input_ring[self.input_pos] = input;
        self.input_pos = (self.input_pos + 1) % self.fft_size;

        let output = self.output_ring[self.read_pos] as f32;
        self.output_ring[self.read_pos] = 0.0;
        self.read_pos = (self.read_pos + 1) % out_len;

        self.hop_counter += 1;
        if self.hop_counter >= self.hop_size {
            self.hop_counter = 0;
            self.process_frame(shift, stretch);
        }

        output
    }

    fn process_frame(&mut self, shift: f32, stretch: f32) {
        let n = self.fft_size;
        let out_len = self.output_ring.len();

        // Extract frame from input ring, apply analysis window
        for i in 0..n {
            let ring_idx = (self.input_pos + i) % n;
            self.fft_buf[i] = Complex::new(
                self.input_ring[ring_idx] * self.analysis_window[i],
                0.0,
            );
        }

        // Forward FFT
        self.fft_forward
            .process_with_scratch(&mut self.fft_buf, &mut self.scratch);

        // Identity short-circuit: skip phase vocoder when no shift/stretch
        let is_identity = shift.abs() < 1e-6 && (stretch - 1.0).abs() < 1e-6;
        if is_identity {
            // Copy fft_buf directly to out_buf (perfect reconstruction)
            self.out_buf.copy_from_slice(&self.fft_buf);
        } else {
            self.remap_bins(shift, stretch);
        }

        // Inverse FFT
        self.fft_inverse
            .process_with_scratch(&mut self.out_buf, &mut self.scratch);

        // Normalize and overlap-add
        let inv_n = 1.0 / n as f32;
        let write_start = self.read_pos;
        for i in 0..n {
            let idx = (write_start + i) % out_len;
            self.output_ring[idx] +=
```

```rust
    fn remap_bins(&mut self, shift: f32, stretch: f32) {
        let n = self.fft_size;
        let half_plus_one = n / 2 + 1;

        // Clear output buffer
        for bin in self.out_buf.iter_mut() {
            *bin = Complex::new(0.0, 0.0);
        }

        let shift_ratio = (2.0_f32).powf(shift / 12.0);

        // Track which source bin contributed the highest magnitude per target bin.
        // This prevents runaway phase accumulation when multiple sources map to
        // the same target (common with stretch < 1.0).
        // We use a temporary array to track the best magnitude per target bin.
        // Reuse: we'll store (best_mag, phase_to_accumulate) per target.
        // Since we can't allocate, we use the existing out_buf as workspace:
        // out_buf[k].re = best magnitude so far, out_buf[k].im = phase increment.
        // After the loop, we construct the final complex values.

        // Phase 1: find best source bin per target
        struct BinContribution {
            mag: f32,
            phase_inc: f32,
        }
        // Stack-allocate for small half_plus_one (2049 for fft_size=4096)
        // Actually, we can't stack allocate a variable-size array. Use out_buf
        // as workspace since we're going to overwrite it anyway.

        // Track best magnitude per target bin using out_buf temporarily.
        // out_buf[k].re = best magnitude, out_buf[k].im = phase increment to use.
        for k in 1..half_plus_one {
            let mag = self.fft_buf[k].norm();
            let phase = self.fft_buf[k].arg();

            // Phase deviation from expected
            let expected_phase_inc = 2.0 * PI * self.hop_size as f32 * k as f32 / n as f32;
            let phase_diff = phase - self.last_input_phase[k];
            let phase_dev = wrap_phase(phase_diff - expected_phase_inc);

            self.last_input_phase[k] = phase;

            // Target bin: stretch first, then shift
            let target_f = k as f32 * stretch * shift_ratio;

            // Linear interpolation: distribute magnitude to two adjacent bins
            let target_lo = target_f.floor() as usize;
            let target_hi = target_lo + 1;
            let frac = target_f - target_lo as f32;

            // Expected phase increment for target bins (correct phase vocoder formula)
            let expected_target_lo = 2.0 * PI * self.hop_size as f32 * target_lo as f32 / n as f32;
            let expected_target_hi = 2.0 * PI * self.hop_size as f32 * target_hi as f32 / n as f32;

            // Phase increment = expected_target + phase_deviation (NOT scaled)
            let phase_inc_lo = expected_target_lo + phase_dev;
            let phase_inc_hi = expected_target_hi + phase_dev;

            // Low bin contribution
            if target_lo > 0 && target_lo < half_plus_one {
                let contrib_mag = mag * (1.0 - frac);
                if contrib_mag > self.out_buf[target_lo].re {
                    self.out_buf[target_lo] = Complex::new(contrib_mag, phase_inc_lo);
                }
            }

            // High bin contribution
            if target_hi > 0 && target_hi < half_plus_one {
                let contrib_mag = mag * frac;
                if contrib_mag > self.out_buf[target_hi].re {
                    self.out_buf[target_hi] = Complex::new(contrib_mag, phase_inc_hi);
                }
            }
        }

        // Phase 2: accumulate phases and construct final complex output
        for k in 1..half_plus_one {
            let mag = self.out_buf[k].re;
            let phase_inc = self.out_buf[k].im;

            if mag > 0.0 {
                self.accumulated_output_phase[k] += phase_inc;
                let out_phase = self.accumulated_output_phase[k];

                self.out_buf[k] = Complex::new(
                    mag * out_phase.cos(),
                    mag * out_phase.sin(),
                );

                // Mirror for negative frequencies
                if k < n / 2 {
                    self.out_buf[n - k] = self.out_buf[k].conj();
                }
            } else {
                self.out_buf[k] = Complex::new(0.0, 0.0);
                if k < n / 2 {
                    self.out_buf[n - k] = Complex::new(0.0, 0.0);
                }
            }
        }

        // DC bin: pass through
        self.out_buf[0] = self.fft_buf[0];
    }
```

```rust
use std::f32::consts::PI;

#[derive(Debug, Clone, Copy, PartialEq, Eq)]
pub enum LfoWaveform {
    Sine = 0,
    Triangle = 1,
    SampleHold = 2,
}

impl LfoWaveform {
    pub fn from_int(v: i32) -> Self {
        match v {
            1 => Self::Triangle,
            2 => Self::SampleHold,
            _ => Self::Sine,
        }
    }
}

    lfo_phase: f64,
    lfo_last_sh_value: f32,
    lfo_sh_rng: u32,
    envelope: f32,
}

impl ModulationState {
    pub fn new() -> Self {
        Self {
            lfo_phase: 0.0,
            lfo_last_sh_value: 0.0,
            lfo_sh_rng: 12345,
            envelope: 0.0,
        }
    }

        self.lfo_phase = 0.0;
        self.lfo_last_sh_value = 0.0;
        self.lfo_sh_rng = 12345;

        self.lfo_phase += phase_inc;

        let wrapped = self.lfo_phase >= 1.0;
        if self.lfo_phase >= 1.0 {
            self.lfo_phase -= self.lfo_phase.floor();
        }

        let phase = self.lfo_phase as f32;

        match waveform {
            LfoWaveform::Sine => (2.0 * PI * phase).sin(),
            LfoWaveform::Triangle => {
                if phase < 0.25 {
                    phase * 4.0
                } else if phase < 0.75 {
                    2.0 - phase * 4.0
                } else {
                    phase * 4.0 - 4.0
                }
            }
            LfoWaveform::SampleHold => {
                if wrapped {
                    self.lfo_sh_rng ^= self.lfo_sh_rng << 13;
                    self.lfo_sh_rng ^= self.lfo_sh_rng >> 17;
                    self.lfo_sh_rng ^= self.lfo_sh_rng << 5;
                    self.lfo_last_sh_value =
                        (self.lfo_sh_rng as f32 / u32::MAX as f32) * 2.0 - 1.0;
                }
                self.lfo_last_sh_value
            }
        }
    }

    /// Process one sample for the envelope follower.
    /// Uses single-pole IIR on absolute value with asymmetric attack/release.
    /// `input_sample` is the raw audio sample (will be rectified internally).
    pub fn tick_envelope(
        &mut self,
        input_sample: f32,
        attack_ms: f32,
        release_ms: f32,
        sample_rate: f32,
    ) {
        let input_level = input_sample.abs();
        let coeff = if input_level > self.envelope {
            1.0 - (-1.0 / (attack_ms * 0.001 * sample_rate)).exp()
        } else {
            1.0 - (-1.0 / (release_ms * 0.001 * sample_rate)).exp()
        };
        self.envelope += coeff * (input_level - self.envelope);
    }

    pub fn envelope_value(&self) -> f32 {
        self.envelope
    }
}
```

```rust
/// Extract per-band average magnitudes from FFT magnitude array.
/// Bands are logarithmically spaced. Returns fixed-size array (no allocation).
pub fn extract_band_magnitudes(mags: &[f32], num_bands: usize, half_plus_one: usize) -> [f32; NUM_BANDS] {
    let mut bands = [0.0_f32; NUM_BANDS];
    if half_plus_one <= 1 || num_bands == 0 {
        return bands;
    }

    let log_min = 1.0_f32.ln(); // = 0.0
    let log_max = (half_plus_one as f32).ln();

    for band in 0..num_bands.min(NUM_BANDS) {
        let lo = ((log_min + (log_max - log_min) * band as f32 / num_bands as f32).exp()) as usize;
        let hi = ((log_min + (log_max - log_min) * (band + 1) as f32 / num_bands as f32).exp()) as usize;
        let lo = lo.max(1).min(half_plus_one - 1);
        let hi = hi.max(lo + 1).min(half_plus_one);

        let mut sum = 0.0_f32;
        let count = (hi - lo).max(1);
        for k in lo..hi {
            if k < mags.len() {
                sum += mags[k];
            }
        }
        bands[band] = sum / count as f32;
    }
    bands
}
```

```rust
    fn process(
        &mut self,
        buffer: &mut Buffer,
        _aux: &mut AuxiliaryBuffers,
        _context: &mut impl ProcessContext<Self>,
    ) -> ProcessStatus {
        let num_samples = buffer.samples();
        if num_samples == 0 {
            return ProcessStatus::Normal;
        }

        let channel_slices = buffer.as_slice();
        if channel_slices.len() < 2 {
            return ProcessStatus::Normal;
        }
        let (first, rest) = channel_slices.split_at_mut(1);
        let left = &mut first[0][..num_samples];
        let right = &mut rest[0][..num_samples];

        let mix = self.params.mix.value() / 100.0;
        let dry_mix = 1.0 - mix;

        let lfo_waveform = modulation::LfoWaveform::from_int(self.params.lfo_waveform.value());
        let lfo_rate = self.params.lfo_rate.value();
        let lfo_shift_depth = self.params.lfo_shift_depth.value() / 100.0;
        let lfo_stretch_depth = self.params.lfo_stretch_depth.value() / 100.0;
        let env_attack = self.params.env_attack.value();
        let env_release = self.params.env_release.value();
        let env_shift_depth = self.params.env_shift_depth.value() / 100.0;
        let env_stretch_depth = self.params.env_stretch_depth.value() / 100.0;

        for i in 0..num_samples {
            let base_shift = self.params.shift.smoothed.next();
            let base_stretch = self.params.stretch.smoothed.next();

            let in_l = left[i];
            let in_r = right[i];

            // Envelope follower (per-sample, stereo average)
            let input_mono = (in_l + in_r) * 0.5;
            self.modulation.tick_envelope(input_mono, env_attack, env_release, self.sample_rate);

            // Dry delay for latency compensation
            let dry_l = self.dry_delay_l[self.dry_delay_pos];
            let dry_r = self.dry_delay_r[self.dry_delay_pos];
            self.dry_delay_l[self.dry_delay_pos] = in_l;
            self.dry_delay_r[self.dry_delay_pos] = in_r;
            self.dry_delay_pos = (self.dry_delay_pos + 1) % self.dry_delay_l.len();

            // Tick LFO at hop boundaries, hold value between hops
            self.hop_counter += 1;
            if self.hop_counter >= HOP_SIZE {
                self.hop_counter = 0;
                self.last_lfo_value = self.modulation.tick_lfo(
                    lfo_waveform, lfo_rate, self.sample_rate, HOP_SIZE,
                );
            }

            let env_val = self.modulation.envelope_value();

            let final_shift = (base_shift
                + self.last_lfo_value * lfo_shift_depth * 24.0
                + env_val * env_shift_depth * 24.0)
                .clamp(-24.0, 24.0);
            let final_stretch = (base_stretch
                + self.last_lfo_value * lfo_stretch_depth * 1.5
                + env_val * env_stretch_depth * 1.5)
                .clamp(0.5, 2.0);

            let wet_l = self.shifter_l.process_sample(in_l, final_shift, final_stretch);
            let wet_r = self.shifter_r.process_sample(in_r, final_shift, final_stretch);

            left[i] = mix * wet_l + dry_mix * dry_l;
            right[i] = mix * wet_r + dry_mix * dry_r;
        }

        ProcessStatus::Normal
    }
```

**Note on hop_counter:** The plugin's `hop_counter` tracks when to tick the LFO. The SpectralShifter has its own internal hop counter for FFT frame processing. These are independent — the LFO ticks at hop boundaries relative to the process() call stream, while the FFT frames trigger based on the SpectralShifter's internal counter.
